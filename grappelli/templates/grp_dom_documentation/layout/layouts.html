Basic Knowledge Layout Containers

Description

Within the {% block content %} you may use one ore more different layouts. Each layout is held by a .grp-layout-container.

In most cases it should be good enough to use one .grp-layout-container within the {% block content %} but one never knows so we made it possible to use different layouts at the same time.

There is a fluid layout, a grid layout and combinations of fluid and grid layouts.

Example

Layout Container 1

.grp-column

Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur. Layout Container 2

In the previous example we use the .grp-layout-containers with <sections /> but you might use them with other elements too.

        <p>Layout Container 1</p>
        ...
    </section>

Fluid Layout Full Width

Description

With a basic fluid layout the .grp-layout-container contains a .grp-column which carries your contents and stretches over the full width.

Example

.grp-column
Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat non proident, sunt in culpa qui officia deserunt mollit anim id est laborum.

Source

In the previous example we use the .grp-layout-container with a <section /> but you might use it with other elements too.


{% block content %}
    <section class="grp-layout-container">
        <div class="grp-column">
            Your contents go in here
        </div>
    </section>
{% endblock content %}
 

Grid Layout24 Columns

Description

With a basic grid layout the .grp-layout-container contains different .grp-column.span-$n which carry your contents.

Example

span-4
span-16
span-4 last
span-8
span-8
span-8 last

Source


{% block content %}
    <section class="grp-layout-container">
        <div class="grp-column span-24">
            <div class="grp-column span-4" />
            <div class="grp-column span-16" />
            <div class="grp-column span-4 last" />
        </div>
        <div class="grp-column span-24">
            <div class="grp-column span-8" />
            <div class="grp-column span-8" />
            <div class="grp-column span-8 last" />
        </div>
    </section>
{% endblock content %}
 

Fluid-Grid Combo Layout Left Fluid, Right Grid

Description

You can use a combination of fluid left column and a right column with a fixed width based on the grid.

To achieve that you have to add two classes to the .grp-layout-container

  • .grp-column-left-fluid
  • .grp-column-right-span-$n where $n is the grid-based width your right column should have

Further more your .grp-layout-container .grp-column-left-fluid .grp-column-right-span-$n has to contain a left and a right column (that's necessary for calculating the layout)

  • .grp-column-left
  • .grp-column-right
