Listings Layout Containers

Description

Within the {% block content %} you may use one ore more different layouts. Each layout is held by a .grp-layout-container.

In most cases it should be good enough to use one .grp-layout-container within the {% block content %} but one never knows so we made it possible to use different layouts at the same time.

There is a fluid layout, a grid layout and combinations of fluid and grid layouts.

In the previous example we use the .grp-layout-containers with <sections /> but you might use them with other elements too.


        {% block content %}
            <section class="grp-layout-container">
                <p>Layout Container 1</p>
                ...
            </section>
            <section class="grp-layout-container">
                <p>Layout Container 2</p>
                ...
            </section>
